# Peach Glow Landing Page

A responsive landing page designed for a skincare product, focused on delivering a clean visual experience and highlighting key product benefits.

## Features

* Responsive design for desktop, tablet, and mobile devices
* Modern and clean user interface
* Smooth scrolling and reveal animations
* Interactive hero section
* Auto-playing testimonials slider with navigation dots
* Animated statistics counters
* Infinite scrolling badges section
* Product gallery with hover effects

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## Project Goals

The goal of this project was to practice front-end development skills by creating a visually appealing landing page that combines responsive design, animations, and user-friendly interactions.
